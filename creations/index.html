# Comprendre Python

Python Enseignement Fullstack Django

Comprendre Python est né de l'intuition que voir la succession d'opérations déclenchées par du code, dans l'ordinateur, est le meilleur moyen de comprendre la programmation. En l'occurrence, la syntaxe enseignée est celle de Python, puisqu'elle est très accessible.
Plus qu'un site internet, Comprendre Python c'est :
  • plusieurs heures de cours, en vidéo et avec des explications détaillées à l'écrit
  • des centaines d'exercices corrigés
  • un design réalisé de zéro et intégré dans un frontend responsive
  • un backend complexe réalisé avec Django (Python)

# JupyThree

Python Open Source 3D Jupyter notebooks threeJS

Un widget Jupyter (Python) pour l'affichage de nuages de points 3D massifs, avec d'excellentes performances interactives. Tandis que les autres librairies offrent une expérience saccadée (zoom et rotation) dès qu'on dépasse les 5000 points, JupyThree reste fluide même avec plusieurs millions de points.
Il gère également les maillages triangulaires.
Basé sur pythreejs et ipywidgets

# ASphere

Fullstack Flask Site internet Intégration de design

ASphere est un organisme de formation en Intelligence Artificielle, qui avait besoin d'un site internet pour présenter ses différents modules.
Le mockup du design a été fourni sous la forme d'une image, que nous avons du intégrer pour le Web et rendre responsive (pour l'adapter aux smartphones). Nous avons également programmé un backend simple (avec Flask, en Python) pour gérer les contenus dynamiques (détails des formations, emailing, etc.)

# Spyrit (à venir)

Python Générateur de site statique Backend Open Source

Ce site (groscot.github.io) est un site statique, c'est-à-dire que le contenu des pages est fixé une fois pour toutes.

Pour le concevoir de manière flexible, nous avons mis au point une librairie Python (Spyrit), qui permet de déclarer des « blocks » et des « composants » réutilisables dans chaque page — pour facilement harmoniser le design. Inspiré du framework Flask, Spyrit ne nécessite pas de programmer un serveur Web : il génère simplement des fichiers .html
